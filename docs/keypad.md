# 6×6 Matrix Keypad Layout

## Base Layer

|       | 1      | 2     | 3    | 4    | 5    | 6    |
|-------|--------|-------|------|------|------|------|
| **A** | SHIFT  | ALPHA | CTRL |  x   | OPTN |  ON  |
| **B** | sqrt   |   ^   |  ²   |  (   |  )   |  lg  |
| **C** | 7      | 8     | 9    | tan  |  DEL | AC   |
| **D** | 4      | 5     | 6    | cos  |  ×   |  ÷   |
| **E** | 1      | 2     | 3    | sin  |  +   |  -   |
| **F** | 0      |  ·    |  EE  | FMT  |  Ans | EXE  |

`ON` : ON / OFF

---

## SHIFT Layer

|       | 1      | 2     | 3    | 4    | 5    | 6    |
|-------|--------|-------|------|------|------|------|
| **A** |        | ALPHA | CTRL | SOLV | MODE |  ON  |
| **B** | cbrt   |  root |  ³   | dot  |  ,   |  ln  |
| **C** | vector | mtrx  | CALC | atan |  INS |  AC  |
| **D** | CONST  | CONV  | MENU | acos |  P   |  C   |
| **E** |  deg   |  rad  |      | asin |  Pol | Rec  |
| **F** |  abs   |   i   |  π   |   e  | PAns |   =  |

---

## CTRL Layer

|       | 1      | 2     | 3    | 4    | 5    | 6    |
|-------|--------|-------|------|------|------|------|
| **A** | SHIFT  | ALPHA |      | STO  | SET  |  ON  |
| **B** | frac   |       | ^-1  |      |  ;   | logab|
| **C** |        |  ↑ Y+ |      | tanh | UNDO | LOCK |
| **D** |  ← X-  |  OK   | → X+ | cosh |  !   | mod  |
| **E** |        |  ↓ Y- |  Z+  | sinh | lcm  | gcd  |
| **F** | ZOOM+  | ZOOM- |  Z-  |  exp | plt3 | plt2 |

`LOCK` : Lock CTRL

---
